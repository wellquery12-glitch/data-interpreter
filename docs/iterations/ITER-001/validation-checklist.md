# ITER-001 用户验证清单

## 一、页面与入口
- [ ] 首页存在“商业建议”入口，点击可进入新页面 `/insights`
- [ ] 新页面可正常加载并展示数据集下拉

## 二、自动分析（销售）
- [ ] 选择有效数据集后，点击“自动分析”可返回结果
- [ ] 结果至少包含 3 条建议
- [ ] 每条建议都包含：发现、证据、建议动作、优先级、影响预估
- [ ] 缺少关键字段时，能返回可分析范围说明而非报错中断

## 三、导出能力
- [ ] 可导出 Markdown 文件
- [ ] 可导出 PDF 文件
- [ ] 导出的内容包含摘要与建议清单

## 四、回归验证
- [ ] 原有控制台问答功能（`/ask`）可正常使用
- [ ] 工具管理页、LLM 配置页可正常访问

## 五、业务确认
- [ ] 当前“销售主题优先”满足本次业务目标
