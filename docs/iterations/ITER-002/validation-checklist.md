# ITER-002 用户验证清单

## 一、主题能力
- [ ] 运营主题可正常触发自动分析并返回结构化建议
- [ ] 财务主题可正常触发自动分析并返回结构化建议
- [ ] 客户主题可正常触发自动分析并返回结构化建议

## 二、结果结构
- [ ] 每条建议包含：发现、证据、建议动作、优先级、影响预估
- [ ] 每个主题结果不少于 3 条建议（字段不足时给出可分析范围说明）
- [ ] 输出内容与所选主题语义一致

## 三、导出与页面
- [ ] 三个主题均可导出 Markdown
- [ ] 三个主题均可导出 PDF
- [ ] 页面加载、分析、导出流程无明显卡顿或异常中断

## 四、回归验证
- [ ] 销售主题能力不回退
- [ ] `/ask` 能力不回退
- [ ] 工具管理页、LLM 配置页可正常访问

## 五、业务确认
- [ ] 主题覆盖满足当前业务范围
- [ ] 输出内容可用于业务评审和行动跟进

## 六、数据仓库验收（2026-03-12 新增）
- [ ] 工作台“选择数据集”与“刷新列表”同级展示“数据仓库”入口
- [ ] 工作台不再展示上传、设置别名、删除数据集功能
- [ ] 工作台选择数据集后，显示元数据总结与推荐分析类型
- [ ] 数据仓库包含“公开数据集 / 私人数据集”双模块
- [ ] 私人数据集支持上传、分类、元数据字段维护（别名/分类/描述/标签/分析说明）
- [ ] 公开数据集支持来源扩展展示，当前 UCI 可检索与导入
- [ ] 已导入的 UCI 数据集再次导入时不重复下载
- [ ] 公开/私人数据集都支持“字段预览”按钮并弹出模态框查看字段与样例

## 七、脚本化验收
- [ ] 执行 `scripts/e2e_repository_workflow.sh`（全链路）
- [ ] 如公网受限，执行 `SKIP_PUBLIC=1 scripts/e2e_repository_workflow.sh`（仅私人链路）
