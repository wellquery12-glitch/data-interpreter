# ITER-002 用户验证清单

## 一、主题能力
- [ ] 运营主题可正常触发自动分析并返回结构化建议
- [ ] 财务主题可正常触发自动分析并返回结构化建议
- [ ] 客户主题可正常触发自动分析并返回结构化建议

## 二、结果结构
- [ ] 每条建议包含：发现、证据、建议动作、优先级、影响预估
- [ ] 每个主题结果不少于 3 条建议（字段不足时给出可分析范围说明）
- [ ] 输出内容与所选主题语义一致

## 三、导出与页面
- [ ] 三个主题均可导出 Markdown
- [ ] 三个主题均可导出 PDF
- [ ] 页面加载、分析、导出流程无明显卡顿或异常中断

## 四、回归验证
- [ ] 销售主题能力不回退
- [ ] `/ask` 能力不回退
- [ ] 工具管理页、LLM 配置页可正常访问

## 五、业务确认
- [ ] 主题覆盖满足当前业务范围
- [ ] 输出内容可用于业务评审和行动跟进
